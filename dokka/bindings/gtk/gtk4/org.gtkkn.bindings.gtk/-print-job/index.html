PrintJob

open class PrintJob(val gtkPrintJobPointer: <Error class: unknown class><<Error class: unknown class>>)(source)

A GtkPrintJob object represents a job that is sent to a printer.

You only need to deal directly with print jobs if you use the non-portable class@Gtk.PrintUnixDialog API.

Use method@Gtk.PrintJob.get_surface to obtain the cairo surface onto which the pages must be drawn. Use method@Gtk.PrintJob.send to send the finished job to the printer. If you donâ€™t use cairo GtkPrintJob also supports printing of manually generated PostScript, via method@Gtk.PrintJob.set_source_file.
